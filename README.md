# ETHARA.AI – Team Task Manager

ETHARA.AI is a modern full-stack task management system where admins can create projects, assign tasks, track progress, and manage teams efficiently.

Built with React, Node.js, Express, MongoDB, and Tailwind CSS.

---

# 🚀 Features

## 🔐 Authentication
- User Signup & Login
- JWT Authentication
- Role-Based Access Control
- Admin & Member Roles

---

# 👨‍💼 Admin Features

## 📁 Project Management
- Create Projects
- Edit Projects
- Delete Projects
- View Project Details
- Track Project Progress

## ✅ Task Management
- Assign Tasks to Members
- Update Task Status
- Delete Tasks
- View All Tasks
- Real-Time Progress Tracking

## 📊 Dashboard Analytics
- Total Projects
- Todo Tasks
- In Progress Tasks
- Completed Tasks

---

# 👨‍💻 Member Features

- Login Securely
- View Assigned Tasks
- Update Task Status
- Track Project Progress

---
